# CSS Gradient Converter 
Bring CSS gradients into Cavalry.

# Long description
Convert CSS linear-gradient syntax into editable gradient shader nodes for your designs.

Supports linear-gradient, radial-gradient, and conic-gradient syntaxes.

Made by your friends in the Canva Creative Team.

# Manual

## Installation
1. Download script file
2. In Cavalry, go to Scripts > Open Script Folder...
3. Drag script into Script folder

## Usage
1. Go to Scripts > CSS Gradient to Cavalry Converter in Cavalry
2. Select the target layer in your composition
3. Paste CSS gradient code (linear-gradient, radial-gradient, or conic-gradient) into the input box
4. Click "Apply Gradient" button
5. Gradient shader node is created and applied

## Example CSS
```css
background: linear-gradient(180deg, #FF6B6B 0%, #4ECDC4 100%);
```

## Support
- Linear gradients with multiple color stops
- Radial gradients with start/end colors
- Conic gradients
- Percentages and hex colors
- Custom angle values
