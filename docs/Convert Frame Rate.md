# Convert Frame Rate
Converts composition frame rate while preserving visual timing of animations and easing curves.

# Long description
Need to adjust your frame rate but don't want to manually adjust your keyframes? This tool adjusts keyframes, layer in/out points and more while preserving easing.

Made by your friends in the Canva Creative Team.


# Manual

## Installation
1. Download script file
2. In Cavalry, go to Scripts > Open Script Folder...
3. Drag script into Script folder

## Usage
1. Go to Scripts > Convert Frame Rate in Cavalry
2. Enter desired frame rate
3. Click Apply to convert

## Features
- Automatic keyframe timing adjustment
- Preserves easing curve shapes during conversion
- Handles multiple attributes/layers simultaneously
- Updates playback range accordingly
