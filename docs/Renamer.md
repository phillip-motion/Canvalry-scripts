# Renamer
Rename assets and layers quickly.

# Long description
Ever need to rename a bunch of layers or assets at once? Use Renamer to prepend, append, find, replace and number them as you please.

Made by your friends in the Canva Creative Team.


# Manual

## Installation
1. Download script file
2. In Cavalry, go to Scripts > Open Script Folder...
3. Drag script into Script folder

## Usage
1. Go to Scripts > Renamer in Cavalry
2. Change settings
3. Hit Apply!


## Modes

### Add
**Prepend:** Text to add before item names.

**Append:** Text to add after item names.

**Apply:** Adds text to all selected items.

**Preview:** Shows original vs new names in table.

### Replace
**Find:** Text to locate in names.

**Replace:** Substitution text.

**Apply:** Replaces all instances in selected items.

**Preview:** Shows affected items and count.

### Number
**Start Number:** Starting value (e.g., 1).

**Padding:** Detects from input (1=0, 01=2, 001=3).

**Position:** Prepend or Append number.

**Reverse:** Reverse numbering order.

**Apply:** Numbers all selected items sequentially.

## Usage
1. Select items
2. Open Renamer (Scripts menu)
3. Choose tab, enter settings
4. Click Apply to execute

## Notes
- Works on assets and layers
- Table preview updates live on setting changes
- Supports hundreds of items efficiently
