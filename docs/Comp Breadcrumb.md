# Comp Breadcrumb 
A dockable toolbar displaying your position in the composition hierarchy, allowing quick navigation to parent compositions

# Long description
Show your position in the composition hierarchy. Navigate up the parent chain by clicking on any parent name.

## Installation
1. Download script file
2. In Cavalry, go to Scripts > Open Script Folder...
3. A toolbar appears showing your current composition and its parents
4. Click any parent name in the breadcrumb to navigate to it
5. The toolbar updates to show the new position

## Features
- Shows current layer name in hierarchy
- Click any parent to navigate up the chain
- Breadcrumb trail shows full path from root to current
- Auto-updates when selection changes
- Dockable interface

## Navigation
1. Select any layer in your composition
2. Breadcrumb refreshes

## Error Handling
- Script shows error message if no valid selection
- Console logs navigation errors for debugging
- Gracefully handles invalid composition hierarchies
