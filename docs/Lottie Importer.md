# Lottie Importer
Import simple Lottie files into your Cavalry scene.

# Long description
Import simple Lottie files into your Cavalry scene. Supports gradients, masks, path animation and easing.

This is an early version, please file a [Github issue](https://github.com/phillip-motion/canvalry-scripts) for any problematic files and we'll improve import over time.

Made by your friends in the Canva Creative Team.


# Manual

## Installation
1. Download script file
2. In Cavalry, go to Scripts > Open Script Folder...
3. Drag script into Script folder

## Usage
1. Go to Scripts > Lottie Importer
2. Select a Lottie .json to import

## Notes
- This is very WIP. Works for simple setups, not others.
