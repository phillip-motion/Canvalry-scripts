# Consolidate Assets 
Bring external assets into your Project Assets folder with automatic categorisation.

# Long description
Assets living outside your Cavalry project? Consolidate Assets scans for all file assets located outside your Project Assets folder and copies them in, relinking them automatically.

Made by your friends in the Canva Creative Team.

## Installation
1. Download script file
2. In Cavalry, go to Scripts > Open Script Folder...
3. Drag script into script folder

## Usage
1. Go to Scripts > Consolidate Assets
2. Click Refresh to scan for missing elements
3. Click Consolidate to copy and relink

## Options

**Categorize by type**
When enabled, assets are placed into subfolders (Audio, Footage, Images, etc.)

**Group assets in Scene Subfolder**
Places all assets in a subfolder with the same title as your filename.

**Exclude reference comps**
Ignores references. Enabled by default.
