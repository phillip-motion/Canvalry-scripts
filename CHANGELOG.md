# Changelog


## [1.2.0] - 2026-04-04

### Added
- Reveal Asset in Finder: simply shows the selected asset in Finder
- Consolidate Assets: copies any external assets into your project assets folder and relinks them automatically


## [1.1.0] - 2026-03-11

### Added
- Lottie Importer: import lottie files into Cavalry! This is an early release, there may be incompatibilities or errors.
- Comp Breadcrumb: Simple breadcrumb to jump quickly to a parent or child comp, similar to Aft*r Effects.

## [1.0.1] - 2026-01-30

### Added

> [!IMPORTANT]
> Easey has moved. [Download the latest version here](https://github.com/sammularczyk/easey/releases/tag/v1.2.0)

-   Localiser: easily localise your Cavalry files using JSON or CSV
-   Find and Replace Text: replace text across your entire file

### Changed
-   Updated script build process
-   Embedded image assets in scripts
